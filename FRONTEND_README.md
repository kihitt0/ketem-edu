# KETEM edu - Образование за рубежом

Интерактивный многостраничный лендинг для учебы за рубежом с полной системой авторизации и управления.

## 🚀 Быстрый старт

Приложение готово к работе! Просто откройте его в браузере.

## 🔐 Демо-доступ

### Студент
- **Email:** student@study.com
- **Пароль:** student123

### Администратор
- **Email:** admin@study.com
- **Пароль:** admin123

## 📁 Структура приложения

### Страницы

1. **Главная (/)** - Hero секция, статистика, преимущества
2. **О нас (/about)** - Миссия, команда, ценности
3. **Мой путь (/goals)** - Пошаговая программа поступления
4. **Соц. сети (/social)** - Социальные платформы и сообщество
5. **Псих. поддержка (/psychology)** - Психологическая помощь студентам
6. **Вход/Регистрация (/login)** - Авторизация и регистрация
7. **Личный кабинет (/dashboard)** - Для студентов (требует авторизации)
8. **Админ-панель (/admin)** - Для администраторов (требует авторизации admin роли)

### Функционал

#### Для студентов:
- ✅ Просмотр своих заявок
- ✅ Отслеживание статуса заявок (В обработке / Одобрено / Отклонено)
- ✅ Рекомендованные программы
- ✅ Статистика профиля
- ✅ Календарь событий

#### Для администраторов:
- ✅ Полное управление заявками (CRUD)
- ✅ Добавление новых заявок
- ✅ Редактирование существующих заявок
- ✅ Удаление заявок
- ✅ Изменение статуса заявок
- ✅ Поиск по студентам, университетам, странам
- ✅ Фильтрация по статусам
- ✅ Статистика (всего, в обработке, одобрено, отклонено)

## 🎨 Дизайн

### Цветовая схема
- **Основной акцент:** Оранжевый (#FF6B35) - используется для кнопок, иконок, ключевых элементов (5-10% страницы)
- **Фон:** Белый и светло-серый
- **Подвал:** Графитовый/черный (#2C2C2C)

### Компоненты
- Адаптивный дизайн для всех устройств
- Sticky навигация
- Мобильное меню
- Модальные окна для управления заявками
- Интерактивные карточки и формы

## 💾 Хранение данных

Приложение использует **localStorage** для:
- Авторизации пользователей (`ketem_user`)
- Зарегистрированных пользователей (`ketem_registered_users`)
- Заявок администратора (`ketem_applications`)
- Заявок студентов (`ketem_student_apps_${email}`)

## 🔄 Регистрация новых пользователей

Вы можете зарегистрировать новых студентов через страницу входа. Все новые пользователи получают роль "student".

## 🛠️ Технологии

- **React 18.3.1** - UI библиотека
- **TypeScript** - Типизация
- **React Router DOM** - Маршрутизация
- **Tailwind CSS v4** - Стилизация
- **Lucide React** - Иконки
- **Vite** - Сборщик

## 📝 Примечания

- Все данные хранятся локально в браузере
- При очистке localStorage все данные будут удалены
- Демо-пользователи (student@study.com и admin@study.com) всегда доступны
- Приложение полностью функционально без backend-сервера

## 🎯 Следующие шаги

Для интеграции с вашим Node.js/Express/PostgreSQL backend:
1. Замените localStorage вызовы на API запросы
2. Используйте JWT токены вместо хранения пользователя в localStorage
3. Подключите реальную базу данных PostgreSQL
4. Добавьте обработку ошибок и валидацию на стороне сервера
