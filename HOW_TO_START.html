🚀 Запуск KETEM EDU Backend

Следуйте этим простым шагам для запуска Socket.IO сервера

1 Откройте терминал (командную строку)

Windows:

  • Нажмите Win + R
  • Введите cmd
  • Нажмите Enter

Mac:

  • Нажмите Cmd + Space
  • Введите Terminal
  • Нажмите Enter
2 Перейдите в папку backend

Выполните команды по очереди:

cd путь/к/вашему/проекту
cd backend

Пример: cd C:\Projects\ketem-edu\backend

3 Установите зависимости (только первый раз)
npm install

Дождитесь окончания (может занять 1-2 минуты)

4 Запустите сервер
npm start
✅ Успех! Вы должны увидеть:
🚀 Socket.IO server running on http://localhost:3000
📡 WebSocket endpoint: ws://localhost:3000
⚠️ Важно!
  • НЕ закрывайте терминал с backend!
  • Backend должен работать одновременно с frontend
  • Держите оба терминала открытыми
5 Проверка работы

Откройте в браузере:

🌐 Открыть http://localhost:3000

Должны увидеть JSON ответ с "status": "OK"

🎉 Готово!
  1. Вернитесь на ваш сайт (http://localhost:5173)
  2. Обновите страницу (F5)
  3. Войдите в систему
  4. Нажмите на кнопку чата 💬 в правом нижнем углу
  5. Чат теперь работает в реальном времени! 🎉
❌ Проблемы?

Ошибка: "npm: command not found"

  • Установите Node.js: nodejs.org
  • Выберите версию LTS (рекомендуемая)

Ошибка: "Port 3000 already in use"

  • Порт 3000 занят другой программой
  • Закройте её или перезагрузите компьютер

Другие ошибки:

cd backend
rm -rf node_modules package-lock.json
npm install
npm start
